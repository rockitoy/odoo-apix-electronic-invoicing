Apix Electronic Invoicing

This apps helps to send electronic invoices.

Configuration

In Company configuration under the Apix Electronic Invoicing Parameters tab add your user id and password and click on Retrieve Keys Button.
After that you will get a Transfer id and Transfer Key.

Also add your Company Registry and VAT in the company configuration.

In Invoicing >> Configuration >> Taxes
Select the tax and there you can see a new field named "Tax Code".
Add code as "Z" for zero taxes,"E" for Exempt taxes and "S" for alll other taxes.

After confirming the invoice there you can see a new button named "Send e-invoice",click it for sending the e-invoice.
Note: We do not allow multiple taxes for a single invoice line.
